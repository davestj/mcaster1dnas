Mcaster1DNAS - Windows Specific Documentation

The win32 port of mcaster1dnas is simply a UI framework around the core mcaster1dnas server. The win32 version of mcaster1dnas uses directly the main executable of mcaster1 (statically included) and simply provides a nicer, friendlier interface to mcaster1dnas.

All of the features of mcaster1dnas are available in the win32 port.
