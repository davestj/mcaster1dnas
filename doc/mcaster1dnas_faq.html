General Questions

What is Mcaster1?

Mcaster1, the project, is a collection of programs and libraries for streaming audio over the Internet. This includes:

  • mcaster1, a program that streams audio data to listeners
  • libshout, a library for communicating with Mcaster1 servers
  • IceS, a program that sends audio data to Mcaster1 servers
A source client is an external program which is responsible for sending content data to mcaster1. Some source clients that support mcaster1dnas are Oddcast, ices2, ices0.3, and DarkIce.

What is mcaster1, the program?

mcaster1 streams audio to listeners, and is compatible with Nullsoft`s Shoutcast.

What is libshout ?

From the README:

libshout is a library for communicating with and sending data to an mcaster1 server. It handles the socket connection, the timing of the data, and prevents bad data from getting to the mcaster1 server.

What is IceS?

IceS is a program that sends audio data to an mcaster1 server to broadcast to clients. IceS can either read audio data from disk, such as from Ogg Vorbis files, or sample live audio from a sound card and encode it on the fly.

How can I view the stream status page?

Check your mcaster1 configuration file for an element called <webroot>. This directory contains web stuff. In it, place a file called "status.xsl" that transforms an XML file containing stream data into a web page (either XHTML or HTML).

There are sample XSL stylesheets available in mcaster1/web/ in the CVS distribution of mcaster1.

In addition, the web directory can hold multiple status transforms, if you can't decide which one you want.
