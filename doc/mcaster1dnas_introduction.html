Introduction to Mcaster1DNAS

Professional streaming media server for modern broadcasting

What is Mcaster1DNAS?

Mcaster1DNAS is a professional streaming media server designed for broadcasting audio content over the internet. It supports multiple audio formats including MP3, AAC, Ogg Vorbis, Opus, Speex, and FLAC, making it one of the most versatile streaming servers available.

Whether you're creating an internet radio station, podcasting platform, or a private streaming service, Mcaster1DNAS provides the robust infrastructure you need. Built on the proven Icecast foundation and enhanced with modern features, it's used by broadcasters worldwide for everything from small personal stations to large-scale professional deployments.

Source Clients

Software that encodes and sends your audio content to the Mcaster1DNAS server. Popular options include Butt, Liquidsoap, SAM Broadcaster, and FFmpeg.

Streaming Server

Mcaster1DNAS receives content from source clients and distributes it to listeners efficiently, handling thousands of concurrent connections.

Listeners

Your audience connects using standard media players like VLC, iTunes, web browsers, or mobile apps to listen to your streams.

What Platforms are Supported?

Mcaster1DNAS runs on all major Unix-like operating systems and supports modern deployment methods including containerization and orchestration platforms.

Linux Distributions

  • Ubuntu (20.04+, 22.04+, 24.04+)
  • Debian (10+, 11+, 12+)
  • Red Hat Enterprise Linux (7+, 8+, 9+)
  • CentOS / Rocky Linux / AlmaLinux (7+, 8+, 9+)
  • Fedora (latest releases)

BSD Unix

  • FreeBSD (12+, 13+, 14+)
  • OpenBSD (latest releases)
  • NetBSD (latest releases)

Other Platforms

  • macOS (10.15+, 11+, 12+, 13+, 14+)
  • Docker (containerized deployments)
  • Kubernetes (orchestrated deployments)
  • Cloud platforms (AWS, GCP, Azure, DigitalOcean)
  • Windows 10/11 — native GUI (mcaster1win.exe) built with Visual Studio 2022 (v17) — see Windows documentation

    • Key Features

    • ICY-META v2.1+ Protocol: Extended metadata — podcast, video, social media fields, and backward-compatible ICY 1.x fallback
    • YAML & XML Configuration: Auto-detected — use either format, both are fully supported
    • Multi-Format Support: MP3, AAC, Ogg Vorbis, Opus, Speex, FLAC, and Theora
    • HTTPS by Default: Dual ports — HTTP on 9300, HTTPS/TLS on 9443, SSL set up in minutes
    • Song History API: In-memory track log at /mcaster1songdata — deduplicated ring buffer, configurable depth
    • Track History UI: Public and admin pages with MusicBrainz, Last.fm, Discogs, and AllMusic lookup icons
    • Built-in Web Player: Browser-based HTML5 player with VU meters, volume control, and keyboard shortcuts
    • Full Codec Stats: Bitrate, sample rate, channels, and codec name publicly exposed in the stats XML
    • Opus ICY Metadata: Now-playing artist/title tracking for Opus streams
    • Advanced Authentication: htpasswd file-based and URL-callback listener authentication
    • Stream Relaying: Master/slave relay with load balancing and geographic distribution
    • Real-time Statistics: Rich XML stats API, modern XSLT web interface, live dashboard
    • Native Windows GUI: Resizable MFC application with CLI flags, auto-versioning, and system tray support
    • YP Directory Logging: Detailed directory registration tracking and health logs
