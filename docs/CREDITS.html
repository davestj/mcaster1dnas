macOS GUI

Qt6 Framework

LGPL 3.0 — qt.io

The macOS GUI was built using the Qt6 toolkit (Qt 6.x from Homebrew). Components used include Qt6 Widgets, Qt6 WebEngineWidgets, Qt6 Network, and Qt6 Xml. Qt is used under the LGPL 3.0 license.

CMake

BSD License — cmake.org

Cross-platform build system used for the Qt6 GUI component. The macos/CMakeLists.txt project compiles all src/*.c server files alongside the Qt6 C++ GUI files into a single in-process binary.

The macOS GUI was built using the Qt6 toolkit (Qt 6.x from Homebrew). Qt is used under the LGPL 3.0 license. Source code for the GUI components is available in the macos/ directory of the repository.

Windows GUI & Installer

Microsoft Foundation Classes (MFC)

Windows GUI Framework

The Windows GUI application (mcaster1win.exe) is built using Microsoft Foundation Classes via Visual Studio 2022 Professional. MFC provides the dialog-based windowing framework, tab controls, and system integration for the Windows desktop experience.

ResizableLib

Resizable MFC Dialogs — Paolo Messina

Open-source library by Paolo Messina providing resizable dialog support for MFC applications. Used to allow the main Mcaster1DNAS Windows dialog to be freely resized while anchoring controls correctly.

NSIS — Nullsoft Scriptable Install System

Windows Installer — nsis.sourceforge.io

Open-source Windows installer framework used to build the Mcaster1DNAS Windows setup executable. The installer script (windows/installer/mcaster1dnas.nsi) handles all installation sections, firewall rules, service registration, and Start Menu / desktop shortcut creation.
